# 내 PC에서 실행하기 — 세팅 가이드

GitHub 서버가 보안에 막혀서, 이제부터는 **내 PC에서 매일 아침 9시에 자동 실행**되도록 바꿉니다.
회사/집 인터넷은 일반 사용자 IP라서 보안 차단에 걸릴 확률이 매우 낮습니다.

## 준비물: Node.js 설치 (최초 1회)

1. https://nodejs.org 접속 → 초록색 **LTS** 버튼 클릭해 다운로드
2. 받은 파일 실행 → 전부 기본값으로 **Next → Install**
3. 설치 확인: 키보드 `Win + R` → `cmd` 입력 → 엔터 → 검은 창에 `node -v` 입력 → 엔터
   → `v22.x.x` 같은 버전이 나오면 성공

## 1단계. 폴더 준비

1. 전달받은 `site-monitor-pc.zip`의 압축을 풀어 **`C:\site-monitor`** 로 옮기세요.
   (한글이나 공백이 없는 경로를 권장합니다)
2. 폴더 안에 이런 파일들이 있어야 합니다:
   `monitor.js, compare.js, report.js, package.json, urls.csv, setup.bat, run.bat`

## 2단계. 최초 설치 (1회만)

`setup.bat` 더블클릭 → 검은 창에서 자동으로 설치가 진행됩니다 (몇 분 소요).
"Setup complete!"가 나오면 아무 키나 눌러 닫으세요.

※ "Windows의 PC 보호" 경고가 뜨면: **추가 정보 → 실행** 클릭

## 3단계. 테스트 실행

`run.bat` 더블클릭 → 사이트 접속/비교/리포트 생성 후 브라우저로 리포트가 자동으로 열립니다.

- 첫 실행은 비교할 "어제"가 없어 전부 "첫 실행"으로 표시됩니다.
- 한 번 더 `run.bat`을 실행하면 있음/없음 판정이 나오는 걸 확인할 수 있습니다.
- 이번에는 스크린샷이 제대로 찍혔는지 `screenshots\오늘날짜` 폴더에서 꼭 확인하세요!

## 4단계. 매일 아침 9시 자동 실행 등록

1. 키보드 `Win` 키 → **"작업 스케줄러"** 검색해서 실행
2. 오른쪽 **기본 작업 만들기** 클릭
3. 이름: `사이트 모니터링` → 다음
4. 트리거: **매일** 선택 → 다음 → 시작 시간 **오전 9:00** → 다음
5. 동작: **프로그램 시작** → 다음 → **찾아보기**로 `C:\site-monitor\run.bat` 선택 → 다음 → 마침
6. (권장) 목록에서 방금 만든 "사이트 모니터링" **더블클릭** → **설정** 탭 →
   **"예약된 시작을 놓친 경우 가능한 대로 빨리 작업 시작"** 체크 → 확인
   → 9시에 PC가 꺼져 있었어도 켜면 바로 실행됩니다.

이제 매일 아침 9시에 자동으로 점검이 돌고, 리포트가 브라우저로 열립니다.
리포트 파일 위치: `C:\site-monitor\reports\index.html`

## urls.csv 수정 방법

`urls.csv`를 **메모장**으로 열어 수정 → 저장.
※ Excel로 편집할 경우 저장할 때 파일 형식을 **"CSV UTF-8"**로 선택해야 한글이 깨지지 않습니다.

## GitHub 저장소는 어떻게 하나요?

이제 실행에는 필요 없습니다. 남겨둬도 되지만, 매일 실패 알림 메일이 오지 않도록 꺼두세요:
저장소 → **Actions** 탭 → daily-site-check → 오른쪽 위 **⋯** → **Disable workflow**

## 문제 해결

- **run.bat 창이 바로 꺼지면서 안 돼요** → `Win+R` → `cmd` → `cd C:\site-monitor` 엔터 → `run.bat` 엔터 → 나오는 오류 메시지를 확인하세요 (스크린샷을 보내주시면 봐드립니다)
- **또 보안에 막힌 사이트가 있어요** → 리포트에 "보안(봇 차단)" 오류로 표시됩니다. 해당 사이트만 알려주세요.
- **PC가 매일 꺼져 있어요** → 4단계 6번 설정을 했다면 PC를 켜는 순간 자동 실행됩니다.
